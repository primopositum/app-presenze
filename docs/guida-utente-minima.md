# Guida Minima Per Impiegato

Questa guida spiega il flusso base giornaliero senza dettagli tecnici.

## 1) Accesso

1. Fai login con email e password.
2. Se il login va a buon fine, puoi compilare presenze e trasferte.

## 2) Compilare le presenze

1. Inserisci la giornata (tipo, ore, data, note se servono).
2. Controlla che ogni giorno sia corretto.
3. A fine mese valida le tue presenze (passaggio a livello utente).

## 3) Gestire una trasferta

1. Crea trasferta con data e azienda.
2. Aggiungi note utili (es. motivo, dettagli cliente, percorso).
3. Carica scontrini (immagini JPG/PNG).
4. Inserisci le spese collegate.

## 4) Generare documenti

1. Puoi generare PDF presenze mese.
2. Puoi generare PDF trasferte mese.
3. Per una persona e un mese specifico puoi generare dossier ZIP completo.

## 5) Cosa non puoi fare (utente non admin)

- Non puoi validare a livello admin.
- Non puoi modificare dati di altri utenti.
- Non puoi modificare o cancellare record gia validati admin.

## 6) Errori frequenti e cosa fare

- Errore permessi: verifica di essere sul tuo utente.
- Errore data: usa sempre formato `YYYY-MM-DD`.
- Errore upload scontrino: controlla formato file (`.jpg`, `.jpeg`, `.png`).
- Errore token/sessione: rifai login.

## 7) Buone pratiche

- Carica scontrini subito, non a fine mese.
- Compila le note quando c'e un caso non standard.
- Controlla il mese prima di validare in blocco.
